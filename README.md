# Calm Elevation

A guided self-healing and emotional regulation app designed to help users stabilize, understand, and gradually elevate their mental and emotional state.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the Project**:
    ```bash
    npx expo start
    ```
    - Scan the QR code with your phone (Expo Go app) or press `a` for Android emulator / `i` for iOS simulator.

## Project Structure

-   `app/`: Application source code and routing (Expo Router).
    -   `index.tsx`: The entry screen.
    -   `_layout.tsx`: Root layout and theme provider.
-   `constants/`: Global configuration and design tokens.
    -   `Colors.ts`: Premium color palette definition.
-   `assets/`: Images, icons, and fonts.
-   `components/`: Reusable UI components (to be populated).

## Tech Stack

-   **React Native** & **Expo** (SDK 52)
-   **Expo Router** for navigation
-   **Expo Linear Gradient** & **Expo Blur** for premium UI aesthetics
-   **TypeScript** for type safety
